Correlation heatmap¶

reference

  1. We generally remove highly correlated features
  2. Reason say if x (input feature), y (target) are highly correlated
  3. We might miss out on some other explanatory feature
  4. Chances are we might not learn the target variable well enough due to this highly correlated feature

ax = sns.heatmap(
    corr,
    xticklabels=corr.columns,
    yticklabels=corr.columns,
    linewidths=0.2,
    cmap="YlGnBu",
)
